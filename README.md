## Prerequisites

Before you begin, ensure you have the following installed:

- Python (version 3.10.4 or higher)
- pip (make sure pip works by running: pip --version)
- [VSCode](https://code.visualstudio.com/)
- [Python Extension for VSCode](https://marketplace.visualstudio.com/items?itemName=ms-python.python)

## Setup Instructions

Follow these steps to set up your virtual environment and install the required packages:

# Step 1: Set up a new environment

python -m venv venv
venv\Scripts\activate (Mac / Linux: source venv/bin/activate)

# Step2: Install packages

pip install -r requirements.txt
